# Day 01 – Programming and Problem Solving

## Objective

The goal of this session is to understand what programming is, why problem solving is important, and how computers execute instructions.

---

## What is Programming?

Programming is the process of writing instructions that a computer can understand and execute.

A program is a sequence of instructions used to solve a problem or perform a task.

---

## What is Problem Solving?

Problem solving is the process of:

1. Understanding the problem
2. Identifying inputs and outputs
3. Designing a solution
4. Implementing the solution
5. Testing the solution

---

## Steps in Problem Solving

### Step 1: Understand the Problem

Read the problem carefully.

### Step 2: Identify Input and Output

Example:

Input:

* Two numbers

Output:

* Sum of the numbers

### Step 3: Design an Algorithm

Algorithm:

1. Read first number
2. Read second number
3. Add both numbers
4. Display the result

### Step 4: Implement in Python

Convert the algorithm into code.

### Step 5: Test

Check the program using different inputs.

## Summary

Programming is a tool for solving problems. Before writing code, it is important to understand the problem and design a clear solution.
