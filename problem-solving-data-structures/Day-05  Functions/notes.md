# Day 05 – Functions

## Objective

The goal of this session is to understand how functions help organize and reuse code.

---

## What is a Function?

A function is a block of code designed to perform a specific task.

Functions help:

* Reduce code repetition
* Improve readability
* Make programs easier to maintain

---

## Defining a Function

Syntax:

```python
def function_name():
    statement
```

Example:

```python
def greet():
    print("Hello")
```

---

## Calling a Function

A function executes only when it is called.

Example:

```python
greet()
```

Output:

```text
Hello
```

---

## Function with Parameters

Parameters allow data to be passed into a function.

Example:

```python
def greet(name):
    print("Hello", name)

greet("Saloni")
```

Output:

```text
Hello Saloni
```

---

## Function with Return Value

A function can return a result using the `return` keyword.

Example:

```python
def add(a, b):
    return a + b
```

Functions divide a program into smaller reusable parts and make problem solving more organized.
