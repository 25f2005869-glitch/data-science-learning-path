# Day 02 – Variables, Data Types and Expressions

## Objective

The goal of this session is to understand how Python stores data using variables and how different data types are used in programs.

---

## Variables

A variable is a named storage location used to store data.

Example:

```python
name = "Saloni"
age = 20
```

Here:

* `name` stores a string value.
* `age` stores an integer value.

---

## Data Types

### Integer (int)

Stores whole numbers.

```python
x = 10
```

### Float (float)

Stores decimal numbers.

```python
y = 10.5
```

### String (str)

Stores text.

```python
name = "Saloni"
```

### Boolean (bool)

Stores True or False.

```python
is_student = True
```

---

## Expressions

An expression combines values and operators to produce a result.

Example:

```python
total = 10 + 20
```

Result:

```python
30
```

---

## Assignment Operator

The assignment operator (`=`) is used to assign a value to a variable.

```python
x = 100
```

---

## Type Checking

Python provides the `type()` function.

Example:

```python
age = 20
print(type(age))
```

Output:

```python
<class 'int'>
```

---

## Key Concepts Learned

* Variables
* Data Types
* Expressions
* Assignment Statements
* Type Checking

---

## Summary

Variables help store data in memory. Different data types are used for different kinds of information, and expressions allow us to perform calculations and manipulate data.
