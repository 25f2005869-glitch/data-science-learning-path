NumPy Matrix Calculator

📌 Project Overview

This project implements a Matrix Calculator using NumPy in Python.
It performs common matrix operations such as addition, subtraction, multiplication, and transpose.

The goal of this project is to demonstrate the use of the NumPy library for efficient numerical computation in Python.

---

🚀 Features

The program supports the following matrix operations:

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose

---

🛠 Requirements

Make sure the following are installed:

- Python 3.x
- NumPy library

Install NumPy using:

pip install numpy
