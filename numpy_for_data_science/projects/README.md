🧮 Matrix Calculator Collection (NumPy + GUI)

📌 Project Overview

This repository contains multiple versions of a Matrix Calculator built using Python and NumPy. It demonstrates matrix operations in both command-line and GUI environments.

The project progresses from a basic implementation to advanced GUI-based applications.

---

🚀 Projects Included

1️⃣ NumPy Matrix Calculator (CLI)

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication

📄 File: "matrix_calculator_numpy.py"

---

2️⃣ Matrix Calculator GUI (Tkinter)

- GUI-based interface
- Matrix Addition, Subtraction, Multiplication
- Easy matrix input

📄 File: "matrix_calculator_gui.py"

---

3️⃣ Advanced Matrix Calculator GUI

- 🎨 Colorful UI
- 📊 Table formatted output
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- 🧮 Determinant
- 🔄 Inverse
- 💾 Save result to file

📄 File: "matrix_calculator_advanced_gui.py"

---

🛠 Requirements

Make sure the following are installed:

- Python 3.x
- NumPy
- Tkinter (comes pre-installed with Python)

Install NumPy:

pip install numpy
