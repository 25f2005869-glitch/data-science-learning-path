# 📖 Overview

This repository contains my complete learning journey in **NumPy**, the fundamental library for numerical computing in Python.

The objective of this repository is to develop a strong foundation in:

* Scientific Computing
* Numerical Analysis
* Data Manipulation
* Data Science
* Machine Learning

The repository follows a structured daily learning approach from NumPy fundamentals to advanced Universal Functions (ufuncs) and practical projects.
