# Python Daily Practice

A structured, day-by-day learning repository focused on **Core Python** and **NumPy for Data Science**.  
Everything is organized into dedicated folders so it stays clean and easy to navigate.

---

## What’s inside

### ✅ Core Python (Fundamentals + Practice)
Folder: `python_practice/`

Covers the main building blocks of Python, including:
- Basics (variables, data types, operators)
- Strings
- Conditionals and loops
- Data structures (list, tuple, dict, set)
- Functions and recursion
- OOP (classes, methods, inheritance)
- File handling
- Exceptions + custom exceptions
- Threading (basics)
- Small practice programs

---

### ✅ NumPy for Data Science
Folder: `numpy_for_data_science/`

Covers NumPy topics such as:
- Array creation, shape, dtype
- Indexing and slicing
- Copy vs view
- Reshape, flatten
- Iteration
- Joining, stacking, splitting
- Searching, sorting, filtering
- Random numbers and distributions
- ufuncs and vectorization
- Mini projects inside `numpy_for_data_science/projects/`

## How to run

### 1) Prerequisites
- Python 3.x installed

### 2) Run Core Python scripts
```bash
python python_practice/<file_name>.py
```

### 3) Run NumPy scripts
Install NumPy first:
```bash
pip install numpy
```

Then run:
```bash
python numpy_for_data_science/<file_name>.py
```

### If a filename contains spaces
Use quotes:
```bash
python "python_practice/<file name with spaces>.py"
```
