# 📊 Matplotlib for Data Science (Daily Practice)

## 📝 Introduction

This repository contains daily Python exercises focusing on data visualization with Matplotlib. It's perfect for beginners, students, and professionals who wish to strengthen their data visualization skills, build portfolio-ready projects, and master plotting basics in small, digestible steps.

## 🛠️ Technologies Used

- Python 3.x
- [Matplotlib](https://matplotlib.org/) (core plotting library)
- [NumPy](https://numpy.org/) (used in some exercises)

---

## ▶️ How to Run

1. **Install dependencies:**
    ```bash
    pip install matplotlib numpy
    ```
2. **Run any practice file:**  
    (If running from project root, use full path)
    ```bash
    python matplotlib_for_data_science/day02_sales_comparison.py
    ```

Output graph images will appear in the same folder.
